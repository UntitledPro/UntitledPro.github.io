【pytorch】：自定义数据集

本文介绍如何使用 pytorch 编写数据集

1. 如何自定义数据集 Dataset

在 pytorch 中编写自定义数据集需要实现三个方法: init, len, getitem.

参照下面的例子， FashionMNIST 的图片都储存在 img_dir 文件夹中，他们的标签都储存在 CSV 文件中(annotation_file)

annotation file:

tshirt1.jpg, 0
tshirt2.jpg, 0
……
ankleboot999.jpg, 9

1.1 init

实例化Dataset对象时，init 函数只运行一次。该函数会初始化数据集的图像，图像对应的注释文件以及Transform（下一节将详细介绍）。

annotation file 的格式如下:

tshirt1.jpg, 0
tshirt2.jpg, 0
……
ankleboot999.jpg, 9

1.2 len

len 函数返回数据集中的样本数。

1.3 getitem

getitem 函数会加载并返回数据集中指定索引的样本。根据索引，它标识图像在磁盘上的位置，使用 read_image 将其转换为张量，从 self.img_labels 中的 csv 数据中检索相应的标签，在其上调用 transform 函数，然后通过字典返回图像的 tensor(张量) 以及相应的标签。

自定义 Dataloader

数据集一次检索一个样本的特征和标签。在训练模型时，我们通常希望以’minibatch’的方式传递样本，在每个 epoch 重新排列数据以减少模型过拟合，并使用 Python 的 multiprocessing 来加速数据检索。

DataLoader是可迭代对象，它通过简单的API为我们抽象了这种复杂性。

Iterate through the DataLoader

We have loaded that dataset into the Dataloader and can iterate through the dataset as needed. Each iteration below returns a batch of train_features and train_labels(containing batch_size=64 features and labels respectively). Because we specified shuffle=True, after we iterate over all batches the data is shuffled (for finer-grained control over the data loading order, take a look at Samplers).
